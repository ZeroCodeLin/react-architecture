## React项目架构
零零散散也做过不少项目了，项目多了总会有一些自己感悟和经验，所以分享出来，肯定有不对和不合适的地方，欢迎批评指正。

### 使用
1. 安装
```
npm install -g yo generator-zero-react
yo zero-react
```
或
``` 
git clone https://github.com/l-Lemon/react-architecture.git
cd react-architecture
npm i
```
2. mock数据模拟
```
npm run mock
```
3. 启动
```
npm start
```
4. 同时启动
```
npm run dev
```
5. 打包构建
```
npm run build
```
